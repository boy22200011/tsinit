# 前言

前置作業
本機要安裝node 
此專案node 版本為 14.16.0
可以用 n 來管理node版本
安裝方式
```
$ sudo npm -g n
```
下載版本
```
$ sudo n install 14.16.0
```
切換版本
```
$ sudo n
```
選擇node版本
確認版本
```
node -v
```

此專案前置
```
$ sudo npm i -g typescript ts-node
```

前置
```
$ sudo npm install
```

執行入口檔
```
$ npx ts-node src/index.ts
```
